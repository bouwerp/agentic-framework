# Agentic Framework Skills

This directory contains specialized skills that extend the capabilities of the agentic framework. Each skill provides instructions and patterns for specific domains.

## Skills Included:
- **aws**: Safe, structured patterns for interacting with AWS services via the CLI.
- **code-review**: A structured post-implementation review for code changes.
- **confluence**: Guidance on interacting with Confluence via the REST API and ADF format.
- **context-compactor**: Techniques for managing context window capacity during complex tasks.
- **figma-interaction**: Universal skill for Figma integration via MCP or REST API.
- **github**: Guidance on GitHub CLI (gh) usage for PR lifecycle management.
- **jira**: Guidance on JIRA REST API interactions using ADF format.
- **planning-with-files**: A methodology for planning and executing complex tasks using persistent markdown files.
- **test-driven-development**: A structured approach to writing tests and implementing code.
